shomescale Architecture

WireGuard full-mesh VPN with central coordination, ACL isolation, DNS, key rotation & web dashboard

Mesh Networking

  • WireGuard P2P tunnels via wg syncconf
  • Full mesh: every node connects to every other
  • 100.64.0.0/10 CGNAT range for internal IPs
  • UDP :51820 on each Pi node
  • 10 Raspberry Pi nodes (Debian 13)

Security Model

  • Server-managed keypair generation
  • UUID-based identity (collision-proof)
  • Symmetric ACL group isolation
  • Key rotation with generation counter
  • Live key update (no restart needed)

Services & Protocols

  • TCP :10000 - Length-prefixed JSON
  • UDP :53 - DNS (*.shomescale)
  • HTTP :8080 - Dashboard + API
  • 60s daemon cycle (heartbeat + sync)
  • dnsmasq local forwarding resolver

Code Modules (12)

  • server.py / client.py - entry points
  • shomescale_protocol.py - framing
  • shomescale_store.py - peer registry
  • shomescale_store_acls.py - ACL engine
  • shomescale_rotation.py - KeyEngine
  • shomescale_dns.py - DNS server
  • 64 pytest tests (100% passing)
